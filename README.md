## 💡 TENTANG PROYEK

DOR ATLANTIC BOT adalah solusi Telegram Bot yang ringkas dan efisien, dirancang khusus untuk mengotomatisasi layanan pembelian paket data XL Provider dan berbagai layanan PPOB (Payment Point Online Bank). Inti dari bot ini adalah integrasi penuh dengan Payment Gateway terkemuka, Atlantic Pedia.

### Fitur Utama
| Ikon | Fitur            | Deskripsi                                                            |
| -----|------------------|----------------------------------------------------------------------|
| 🔑   | Manajemen Akun   | Sistem Login dan pendaftaran pengguna bot yang aman.                |
| 💰   | Top Up Otomatis  | Proses pengisian saldo PPOB yang cepat dan terintegrasi.            |
| 📦   | Transaksi PPOB   | Pembelian pulsa, paket data XL, token listrik, dan layanan PPOB lainnya. |
| ⚙️   | Telegram API     | Interaksi real-time dengan pengguna melalui Telegram Bot API.       |

## 🚀 INSTALASI CEPAT

Untuk menjalankan versi dasar dari DOR ATLANTIC BOT, ikuti langkah-langkah instalasi otomatis berikut:

Unduh script instalasi
```
wget https://github.com/mudziboy/doratlantic/raw/refs/heads/main/install-bot
```

Berikan izin eksekusi

```
bla bla bla
```

Jalankan instalasi

```
pyhton3 main.py
```

Untuk Restart Bot

```
sudo systemctl restart dor-xl-bot.service
```
Untuk Cek log

```
journalctl -fu dor-xl-bot.service
```

> ⚠️ **Perhatian Penting**  
> Script di repositori ini masih dalam tahap pengembangan aktif (Under Development). Beberapa fungsi mungkin memerlukan penyesuaian, konfigurasi manual, dan proses debugging untuk dapat berjalan optimal. Jika ingin Script Full langsung pakai, hubungi kontak dibawah!
